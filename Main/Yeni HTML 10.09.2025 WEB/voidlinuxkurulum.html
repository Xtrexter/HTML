Void Linux Nedir?

Void Linux, bağımsız olarak geliştirilen, hızlı, hafif ve minimalist bir GNU/Linux dağıtımıdır. Geleneksel init sistemleri yerine runit kullanarak son derece hızlı açılış süreleri sunar. Paket yöneticisi olarak XBPS (X Binary Package System) kullanır ve rolling release modelini takip eder.

ISO Seçenekleri

Normal bir bilgisayar için x86_64 kısmından indireceğiz. Eğer minimal kurulum yapılacaksa base seçilebilir ama arayüz ile kurmak isterseniz XFCE tavsiyemdir.

Glibc vs Musl

Glibc (GNU C Library)

Çoğu Linux dağıtımında kullanılan standart kütüphane. Daha fazla program uyumluluğu sağlar, geniş bir ekosisteme sahiptir. Ancak musl'a göre biraz daha büyük ve ağırdır.

Musl

Daha hafif, minimalist ve güvenlik odaklı alternatif. Küçük sistemler ve düşük kaynak tüketen sistemler için ideal. Bazı yazılımlar için ekstra uğraş gerekebilir.

Bu rehberde Glibc XFCE ISO dosyası ile devam edeceğiz.

🚀 Kuruluma Başlama

Bu ekranda ilk seçenek ile devam edin. Ama siyah ekranda kalıyorsa 3. seçenekteki "Graphics Disabled" olanı seçebilirsiniz.

1️⃣ Root Hesabına Geçme

XFCE masaüstü ortamı geldiğinde terminal'i açın.

sudo su

Eğer normal hesap ile devam edecekseniz VoidLinux'un Live şifresi: "voidlinux"

2️⃣ Kurulumu Başlatma

Terminal'e şunu yazarak kuruluma geçin:

void-installer

Kurulum Adımları

Keyboard (Klavye)

Aşağıya inerek istediğiniz klavye düzenini seçin. Türkçe Q Klavye için trq yazanı seçin.

Network (Ağ)

Bağlantı durumunu gösterir. Çalışıp çalışmadığını kontrol edip "Tamam" diyerek geçin.

Source (Kaynak)

Local

ISO'nun içindeki paketleri ana sisteme yükler. İnternetiniz kısıtlı ise kullanılabilir.

Network (Önerilen)

Paketleri ağ üzerinden yükler. Tüm paketler güncel gelir.

Bu rehberde Network (Ağ üzerinden kurulum) yapacağız.

Mirror (Yansı)

Size en yakın yansıyı seçmek çok önemlidir. EU/Finland yada EU/Turkey gibi yakın bir sunucu seçebilirsiniz.

Temel Ayarlar

  • Hostname: Sisteme vereceğiniz isim
  • Timezone: Türkiye için Europe/Istanbul
  • Root Password: Root şifrenizi belirleyin
  • User Account: Kullanıcı hesabınızı oluşturun

Bootloader (Önyükleyici)

VoidLinux'u kuracağınız disk/bölümü seçin. "Use Graphical Terminal For Bootloader" sorusuna yes deyin.

Disk Bölümlendirme

🔨 Partition Aracı Seçimi

cfdisk hem kolay hem kullanışlıdır. cfdisk'i seçin.

UEFI sisteme kurmak için GPT seçin.

Bölüm Planı

3 adet alan ayıracağız:

Bölüm Boyut Tip Mount Point
EFI (BOOT) 512M EFI System /boot/efi
SWAP 2GB (RAM'e göre) Linux Swap -
ROOT (/) Kalan alan Linux filesystem /

➕ Bölüm Oluşturma

  1. Alt kısımda New'e basın
  2. EFI için 512M yazın
  3. Free Space'e tıklayıp SWAP için 2G yazın
  4. Kalan alanı ROOT için ayırın (otomatik)

🏷️ Bölüm Tiplerini Ayarlama

  • /dev/sda1 (EFI): EFI System
  • /dev/sda2 (SWAP): Linux Swap
  • /dev/sda3 (ROOT): Linux filesystem

Son halinde Write ile kaydedin ve Quit ile çıkın.

Dosya Sistemi Oluşturma

FileSystem Ayarları

  • EFI (/dev/sda1): vfat (FAT32) → Mount: /boot/efi
  • SWAP (/dev/sda2): Linux Swap
  • ROOT (/dev/sda3): btrfs veya ext4 → Mount: /

Her dosya sistemi oluşturması için gelen soruya evet deyin.

Kurulumu Tamamlama
